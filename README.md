# Ardent Prayers

## Overview
Ardent Prayers is a digital wall for people to post their prayer needs.

## Technology
+ AWS
    + API Gateway
    + CloudFront
    + DynamoDB
    + Lambda
    + Route 53
    + S3
+ CSS3/SCSS
+ HTML5
+ JavaScript
+ Node.js
+ NPM
+ React
+ Vite

## Install
```bash
$ cd ardent-prayers
$ npm install
```

## Run
```bash
$ npm run dev # dev
$ npm run build # prod
```
